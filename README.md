# python-practice-programs

This repository contains basic Python programs created to practice core programming concepts such as loops, conditionals, string manipulation, and simple logic building.


Programs Included:

HCF Calculator – Finds the highest common factor of two numbers
Number Palindrome – Checks if a number is a palindrome
String Palindrome – Checks if a string is a palindrome
Pattern Programs – Prints star and pyramid patterns
Vowel Counter – Counts vowels in a given word
Character Pattern – Prints patterns using characters
Rock Paper Scissors – Simple command-line game
To-Do List App – Basic command-line task manager



Topics Covered:

These programs demonstrate the following Python concepts:

Variables and data types
Conditional statements (if, elif, else)
Loops (for, while)
Nested loops
String manipulation
Functions
Lists and dictionaries
Basic algorithms and logic building
User input and output


This repository is part of my journey to strengthen Python fundamentals and problem-solving skills before moving to advanced topics like data analysis and machine learning.
