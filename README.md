# python-practice-programs
Python Practice Programs

This repository contains Python programs that I created while learning Python programming.

Programs Included
1. Star Pattern Program

Prints star patterns using loops and conditional statements.

2. Rock Paper Scissors Game

A simple command-line game where the user plays Rock Paper Scissors against the computer.

3. Character Creator

A small RPG-style character generator that validates character stats such as strength, intelligence, and charisma.

Concepts Used

Python loops (for, while)

Conditional statements (if, elif, else)

Functions

Input validation

String manipulation
